# Offset Lab

Offset Lab is a Tesla Model 3 fitment MVP built with Next.js, TypeScript, and Tailwind CSS.

## Included
- Homepage / entry screen
- Fitment recommendation flow
- Compare setup page
- Tesla Model 3 RWD, Long Range AWD, and Performance baselines
- OEM+, Flush, and Aggressive presets
- Poke, inner-clearance, and tire-diameter calculations

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes
- Data is seeded for MVP planning and should be verified before public launch.
- Recommendation logic is enthusiast-first, not purely conservative.
